Towards an Algorithm for the Multi-Source Multi-Sink Steiner Network Problem in the Plane

Description:

The undirected Steiner tree problem within the Euclidean plane is effectively solved in practice by the geosteiner algorithm via an efficient pruning mechanism. A directed variant of the problem exists, where a set of sources, and sinks are given and one must form a network to connect all sources to all sinks.By simple reduction, one finds the problem of the Multi-Sink Steiner Network Problem to be similarly NP-Hard. In this project, we explore some combinatorial aspects of the problem, and seek to test a pruning heuristic and the types of subnetworks which can be effectively pruned, with respect to a convexity measure.
